Great Indian Grocery – QA Validation Plan

Scope
This document outlines functional and regression validation performed during product implementation and UX restructuring.

1. Homepage Validation
- Verify “Shop Now” redirects correctly to Departments section
- Confirm homepage layout matches updated UX design
- Validate promotional modules render correctly
- Check missing department icons are restored

2. Department & Category Validation
- Validate Organic Foods section placement
- Confirm category taxonomy updates reflect correctly
- Verify product-to-category mapping integrity
- Validate SKU visibility and product metadata accuracy

3. Checkout & Pickup Workflow
- Confirm curbside pickup option appears correctly
- Validate car make/model and color metadata capture
- Confirm checkout flow stores metadata properly
- Verify session persistence through checkout process

4. Promotional Campaign Logic
- Confirm weekly promotion timer resets correctly
- Validate backend configurability
- Verify frontend display consistency

5. Content Validation
- Confirm renaming of sections (Customer Favorites, New Arrivals)
- Validate About Us content updates
- Confirm phone number is clickable
- Verify social media redirects

6. Regression Checklist
- Cart add/remove behavior
- Quantity updates
- Checkout completion
- Mobile responsiveness
- Session continuity validation

Status
Validation ongoing – aligned with iterative release cycles.
